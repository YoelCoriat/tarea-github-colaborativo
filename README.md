# Repositorio colaborativo en Python
Enunciado:
Crea un repositorio colaborativo en GitHub donde el equipo de desarrollo pueda trabajar de manera conjunta en un proyecto en Python. El proyecto debe incluir al menos tres módulos distintos con funcionalidades separadas, y cada miembro del equipo deberá contribuir a la creación y mejora de alguna de las partes del código.

Configuración del repositorio: Inicializa un repositorio nuevo en GitHub y configura el entorno de desarrollo adecuado.

Invita a los miembros de su grupo

Desarrollo de módulos(archivos): Crea y organiza tres módulos con funcionalidades específicas: un módulo para operaciones matemáticas, Otro de impresión de un cuento

Documentación: Incluye un archivo README con las instrucciones de uso del repositorio y la descripción de cada módulo.

Colaboración: Cada miembro del equipo debe realizar al menos un "pull request" (enviar y modificar el código de su compañero) y hacer revisiones de código de otros miembros.
